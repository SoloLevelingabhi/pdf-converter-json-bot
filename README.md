
# PDF to JSON Converter Bot

🤖 A powerful Telegram bot that converts SSC exam PDFs to structured JSON format with real-time progress tracking.

## ✨ Features

- 📄 Extract questions from SSC exam PDFs
- 🔍 Real-time progress tracking with ETA
- 📊 Automatic question numbering and formatting
- 🎯 Preserves original wording and formatting
- 📈 Export to structured JSON format
- 🔄 Support for multiple question types
- 📋 Detailed conversion reports
- 🐛 Comprehensive error tracking and debugging
- 💾 MongoDB integration for analytics
- 🐳 Docker support for easy deployment
